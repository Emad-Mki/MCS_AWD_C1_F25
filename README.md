## ✨ المميزات الرئيسية

- عرض الدورات بنظام بطاقات مع ترميز لوني لحالة التوفر
- لوحة تحكم لإضافة / تعديل / حذف الدورات
- دعم رفع الصور يدويًا أو جلبها تلقائيًا عبر Unsplash API
- وضع ليلي / نهاري محفوظ في المتصفح
- تصميم متجاوب (Responsive) لجميع الشاشات
