## 🚀 طريقة تشغيل المشروع

### 1. تشغيل المشروع محليًا (Local)

1. تأكد من وجود Python 3 وبيئة عمل افتراضية (اختياريًا).
2. استنسخ المشروع من GitHub:

   ```bash
   git clone https://github.com/Emad-Mki/MCS_AWD_C1_F25.git
   cd MCS_AWD_C1_F25
   ```

3. ثبّت المتطلبات:

   ```bash
   pip install -r requirements.txt
   ```

4. حدّث إعدادات قاعدة البيانات في `app.py` أو ملف الإعدادات حسب الهيكل المستخدم.
5. قم بتشغيل التطبيق:

   ```bash
   python app.py
   ```

6. افتح المتصفح على العنوان:

   ```text
   http://localhost:5000
   ```

> ملاحظة: عند التشغيل محليًا، لا تحتاج إلى أي إعدادات خاصة بـ PythonAnywhere، فقط تأكد من إعداد الاتصال بقاعدة البيانات بشكل صحيح.

---

### 2. تشغيل المشروع على PythonAnywhere (Deploy)

المشروع حاليًا مرفوع وجاهز على PythonAnywhere عبر الرابط أعلاه، لكن فيما يلي خطوات عامة في حال إعادة نشره على حساب آخر [web:3][web:10]:

1. إنشاء حساب على PythonAnywhere وتسجيل الدخول.
2. رفع ملفات المشروع (عن طريق Git أو Upload من واجهة Files).
3. إعداد بيئة افتراضية وتنصيب المتطلبات:

   ```bash
   pip install -r requirements.txt
   ```

4. إنشاء Web App من تبويب **Web** واختيار:
   - Manual configuration
   - نفس إصدار Python المستخدم في المشروع

5. ربط الويب آب بملف WSGI وضبط:
   - مسار المشروع (app.py)
   - مسار الـ virtualenv

6. إعادة تحميل (Reload) التطبيق من تبويب **Web**، ثم الدخول للرابط العام الذي تولّده PythonAnywhere.

## ✨ المميزات الرئيسية

- عرض الدورات بنظام بطاقات مع ترميز لوني لحالة التوفر.
- لوحة تحكم لإضافة / تعديل / حذف الدورات بشكل ديناميكي.
- دعم رفع الصور يدويًا أو جلبها تلقائيًا عبر Unsplash API.
- وضع ليلي / نهاري محفوظ في المتصفح باستخدام localStorage.
- تصميم متجاوب (Responsive) يعمل على الحاسوب والجوال.
